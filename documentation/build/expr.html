t_test(original_measurements, measurements_after_alteration)[source]¶

This is the paired T-test (for repeated measurements): Given two sets of measurements on the SAME data points (folds) before and after some change, Checks whether or not they come from the same distribution.

This T-test assumes the measurements come from normal distributions.

Returns:
The probability the measurements come the same distributions A flag that indicates whether the result is statically significant A flag that indicates whether the new measurements are better than the old measurements

class BestProbaForDecision¶

class experiments.BestProbaForDecision(dir_name, test=False)[source]¶

An experiment to test what is the best threshold from which we want to make the decision.

__init__(dir_name, test=False)[source]¶
_no_detail¶

Reporting on low verbosity - only tables

run()[source]¶

For each probability p in [0.34,0.59] (in jumps of 0.01), make the decision only if classifier’s probability is greater or equal to p.

For each probability we calculate the amount of games that qulified and the score will be calculated by this amount.
